/*
	About prefixes:
		There are three [DR] prefixes and each one is used with different types of messages.
		Some are used as feedback to the player for their commands and settings changes.
		Some are used for general information and some are used as announcements.
		I feel this is useful for providing the player with context and in helping to identify
		important messages on servers with a lot of chat.
		
	About colour:
		The colour of chat is set by inserting an ASCII control character and a six or eight character
		hexadecimal value before the chat. When using eight characters, the last two are for
		alpha transparency. You can type the control characters in Windows by holding Alt and pressing
		specific numerical key pad keys. https://www.ascii-code.com/
		
			Alt + 01				Used to return to the default colour.
			Alt + 07				Used with a six character hex value.
			Alt + 08				Used with an eight character hex value.
			
		An example phrase:
		
			"Your name is 87CEEB{1} and you have 87CEEB{1} points"	
*/

/*
	Phrases which may need to be translated in future:
		Replies to the command to set your language to English.
		Reply to the debug command /data.
	
	Phrases which have been modified since they were translated will have 'modified' in a comment after them.
*/
